Deploy blog on github with hexo

Apr 14, 2016 | Hits

This is a simple tutorial on how to deploy your own blog on github using node and hexo.

I was inspired by the blog-talking event of Bittiger and decided to start writing blogs. Blog is a good approach for me to strengthen the knowledge I gained and share it with others.

Hexo is a great tool to write blog with markdown language, it is powerful but simple and easy to pick up. I feel that it would be great if I could have my blog deployed on github with the domain myusername.github.io. It is helpful if you point out that you are running and writing a blog on resume.

There are already tons of good tutorials on how to build up a hexo blog, so for this tutorial, my foucs will be simple, how to config and deploy the blog on your github page. If you want deploy blog on your own domain, I believe it is super easy to find a tutorial that teches how to do that.
